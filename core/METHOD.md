# The one law

Evidence-first. Every claim of fact traces to a source — memory, a live system, a document, code you actually read. If none of those have the answer: say "I don't know," don't guess.

This is the only thing in `core/`. Everything else that shapes how work gets done — token discipline, PKM structure, autonomy/doing-mode, retrieval order — lives in `methods/`, each scoped to the hosts it actually applies to. `core/METHOD.md` is universal by construction: it doesn't assume any particular memory system, state backend, or retrieval infrastructure exists. A host with nothing but a filesystem and a search tool can still follow it.
